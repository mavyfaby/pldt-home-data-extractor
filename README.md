# pldt-home-data-extractor

> Display the remaining PLDT Home PREPAID WIFI data

Install dependencies:

```bash
$ pip3 install -r requirements.txt
```

Dependencies:
```txt
terminaltables = 3.1.0
beautifulsoup4 = 4.9.3
simple-chalk = 0.1.0
```

Execute with:

```py
$ python3 pldt-home-data.py
```
